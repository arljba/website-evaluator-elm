Info

Domain

Select this if you want some general information about the host of a website. This is similar to the whois terminal command. This might not work with your dountrys domain ending e.g .de .

Speed

This will fetch data about the speed of a website. How quickly stuff is loaded or the server takes to respond

Stack

Selecting this will give you information about the technology stack used on a given website. For example if the wesite uses an apache webserver and what cms is used on it etc.

Structure

This will collect data about how pages on a website are connected and visualizes them with a directed graph. This will also give feedback if all links on a given site are working or not.
